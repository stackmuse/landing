# StackMuse Landing

Landing page for [StackMuse.com](https://stackmuse.com/)

## Requirements

1. [Node.js](https://nodejs.org/en/download/package-manager/current)

## Development

### Clone

```shell
git clone https://github.com/stackmuse/landing
```

### Install

```shell
npm install
```

### Scripts

```shell
npm run compile
npm run watch
```
